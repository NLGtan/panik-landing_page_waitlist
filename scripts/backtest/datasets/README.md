# PANIK backtest datasets (CSV)

Generated by `node scripts/backtest/export-csv.mjs` from the reconstructed backtest data.
All numbers are real (on-chain liquidations + exact health factors); see
`docs/technical-docs/BACKTEST_OVERVIEW.md` for method.

## Black-swan headline files
- **`black_swan_events.csv`** — the crises we backtested: liquidations, wallets, USD, peak
  S_asset_risk, and whether the crash-regime gate fired.
- **`backtest_results.csv`** — validation outcome per event/protocol: baseline vs shipped recall,
  false-alarm, precision, median lead time. (Price-walk rows have recall+lead only, no false-alarm.)

## Raw per-wallet evidence
- **`positions_<event>.csv`** (`june`, `ust`, `ftx`, `usdc`, `aave-aug24`) — long format, one row per
  (wallet, sampled block): `owner, liquidated, first_liq, block_label, block_time, health_factor`.
  Exact HFs from archive RPC. `health_factor` blank = no debt at that block.
- **`liquidations_<protocol>_aug2024.csv`** (`compound`, `moonwell`, `morpho`) — liquidated borrowers
  on Base in the Aug-2024 crash: `owner, first_liq` (price-walk cohorts).

## Price series
- **`prices_weth_hourly_aug2024.csv`** — hourly WETH/USD around Aug-5-2024.
- **`prices_weth_wbtc_daily_aug2024.csv`** — daily WETH & WBTC, May–Aug 2024 (asset-risk inputs).

(June/UST/FTX/USDC Ethereum price fixtures live in `packages/scoring/tests/fixtures/`.)

Regenerate any time: `node scripts/backtest/export-csv.mjs`.
